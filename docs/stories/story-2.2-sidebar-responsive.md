# Story 2.2 — Sidebar Responsive + Mobile Navigation

**Epic:** EPIC-TD-001
**Sprint:** 2
**Estimativa:** 20 horas
**Prioridade:** CRITICAL

---

## Objetivo

Tornar a navegacao da plataforma utilizavel em mobile (< 768px), com hamburger menu para instructor/admin e bottom tabs para students.

## Tasks

- [ ] **T1: Hamburger menu trigger** (4h) — Botao hamburger no Header para mobile
- [ ] **T2: Sidebar como drawer** (8h) — Slide-in overlay em mobile, fechar ao clicar fora
- [ ] **T3: Bottom tabs para students** (6h) — Dashboard, Cursos, Conquistas, Perfil
- [ ] **T4: Search mobile** (2h) — Icone de search que expande no Header mobile

## Criterios de Aceite

- [ ] Em 375px: sidebar nao visivel, hamburger menu visivel
- [ ] Em 768px+: sidebar visivel normalmente
- [ ] Students: bottom tabs com 4 items funcionais
- [ ] Search acessivel em qualquer viewport
- [ ] Drawer fecha ao clicar fora ou Escape
